# Aanya Agarwal — Personal Portfolio

**Live:** https://personal-portfolio-seven-tau-30.vercel.app/

Personal portfolio website for Aanya Agarwal, CS student at Georgia Institute of Technology. Built with vanilla HTML, CSS, and Font Awesome.

## Pages

| Page | Description |
|---|---|
| **About** | Hero section, bio, skills grid, featured projects |
| **Resume** | Education, experience, research, skills — full resume |
| **Projects** | All projects organized by Full-Stack/AI, ML Research, and Web Dev |
| **Contact** | Contact form and info panel |

## Tech

- HTML5 / CSS3 (Flexbox, Grid, custom properties)
- Font Awesome 6 icons
- Google Fonts (Noto Serif)
- Deployed on Vercel

## Run Locally

```bash
# No build step needed — open directly in a browser
open index.html
```

Or serve with any static file server:

```bash
npx serve .
```
